# Arogya-Nidhi-Traders
A premium e-commerce website for Arogya Nidhi Traders, showcasing traditional Vedic Bilona Cow and Buffalo Ghee. Built with HTML, Tailwind CSS, and JavaScript, featuring an interactive product catalog, shopping cart, checkout flow, WhatsApp ordering integration, QR-based payments, and a modern heritage-inspired user interface.

# 🥛 Arogya Nidhi Traders

**Pure Vedic Bilona Ghee | Traditional Heritage | Modern E-Commerce**

A premium, responsive e-commerce website for **Arogya Nidhi Traders**, dedicated to delivering authentic **A2 Bilona Cow Ghee** and **Buffalo Ghee** prepared using traditional Vedic methods.

The website combines a modern user experience with India's rich dairy heritage, allowing customers to explore products, learn about the Bilona process, generate payment QR codes, and place orders seamlessly through WhatsApp.

## 🚀 Deployment

### GitHub Pages

1. Upload files to GitHub repository.
2. Open Repository Settings.
3. Navigate to Pages.
4. Select:

   * Source: Deploy from a branch
   * Branch: main
   * Folder: / (root)
5. Save changes.
6. Wait a few minutes for deployment.

### Vercel (Recommended)

1. Push repository to GitHub.
2. Connect GitHub account to Vercel.
3. Import repository.
4. Deploy instantly.
5. Connect custom domain.
